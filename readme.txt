== Description ==
This plugin insert a facebook login in your login page.
It's easy to install.
You only insert your facebook code.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:info@nsthemes.com">info@nsthemes.com</a>.*


== Installation ==

1. Upload the entire 'ns-social-login' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Custom Checkout' and flag fields to hide
